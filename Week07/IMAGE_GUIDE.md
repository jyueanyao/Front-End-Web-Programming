## 如何添加圖片

1. **保存圖片**：
   - 右鍵點擊附件中的圖片
   - 選擇 "另存為" 或 "保存圖片"
   - 使用上述指定的檔名保存

2. **放置位置**：
   ```
   Week08/
   └── img/
       ├── thrive-aha.jpg     ← 新增
       ├── easytap.jpg        ← 新增  
       ├── TechCare.png       ← 已存在
       └── Recycle.png        ← 已存在
   ```

3. **檢查結果**：
   - 重新載入 `index.html`
   - 確認所有項目圖片都能正常顯示

## 圖片規格建議

- **格式**: JPG 或 PNG
- **尺寸**: 建議 800x600 像素或更高
- **大小**: 建議小於 2MB 以確保載入速度
- **品質**: 清晰可讀，適合網頁顯示

## 備選方案

如果無法保存原始圖片，您也可以：
1. 截圖保存您的項目展示圖
2. 使用項目的代表性圖片
3. 臨時使用現有圖片進行測試

添加圖片後，您的項目展示就會顯示正確的視覺內容！
